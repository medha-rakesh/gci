# Grokking the Coding Interview: Patterns for Coding Questions

A complete offline archive of the [Educative.io](https://www.educative.io) course *Grokking the Coding Interview: Patterns for Coding Questions*, saved as standalone HTML files and organized by chapter and problem.


Each chapter folder contains subfolders per problem. Each problem subfolder contains up to 4 HTML files — one per language:

| Prefix | Language |
|--------|----------|
| `1.1` | Java |
| `1.2` | Python |
| `1.3` | C++ |
| `1.4` | JavaScript |

The HTML files are self-contained pages saved with [SingleFile](https://github.com/gildas-lormeau/SingleFile) from Educative.io.

## How to read the course

### Option 1 — Viewer (recommended)

1. Clone or download this repo
2. Open `gci_viewer.html` in your browser
3. Browse all 19 chapters from the sidebar, switch languages with the tab bar, and navigate with Prev / Next

> The viewer fetches files live from GitHub so you need an internet connection. The repo must be public.

### Option 2 — Open files directly

Navigate into any chapter/problem folder and open an HTML file directly in your browser. Each file is fully self-contained.

## Viewer features

- **Sidebar** — all 19 chapters collapsible, click any problem to load it
- **Language tabs** — Java / Python / C++ / JS switcher per problem
- **Prev / Next** — step through the entire course in order
- **Search** — filter problems by name across all chapters
- **Progress counter** — shows current page out of total
- **↗ Open** — opens the raw file in a new tab if the viewer can't render it
